# LMSEDSRp
This is the code for our paper "Data-Driven Magnetic Core Loss Modeling with Multiple Variables Crossing Materials". It will be refined upon the paper's publication.

The dataset can be obtained from IEEE DataPort (Zengrui Yi, "The processed MagNet dataset for magnetic core loss modeling", IEEE Dataport, July 28, 2025, doi:10.21227/qgyg-q357)
